# 🏏 IPL 2024 Analytics Dashboard — Power BI

## Dashboard Preview
![IPL 2024 Dashboard](IPL_2024_Dashboard_Preview.png)

## Overview
Interactive IPL 2024 season dashboard built in Power BI using real ball-by-ball match data from Kaggle. Covers all 71 matches of the season with player, team, and match-level insights.

## Key Metrics
- 🏆 Total Matches: 71
- 🏅 Orange Cap: V Kohli — 741 runs
- 🎯 Purple Cap: HV Patel — 30 wickets
- 🥇 Champions: Kolkata Knight Riders — 11 wins

## Visuals Included
- KPI Cards — Total Matches, Total Runs, Top Scorer, Purple Cap
- Team Wins Horizontal Bar Chart
- Toss Decision Donut Chart
- Top Batsmen Table (run-wise sorted)
- Top Bowlers Table (wicket-wise sorted)
- Run Trend Area Chart (match-wise)

## Tools & Technologies
- Power BI Desktop
- Microsoft Excel (data source)
- Python — Pandas (data cleaning & preprocessing)

## Data Source
Kaggle IPL Dataset
- matches.csv — 1,095 matches (2008–2024)
- deliveries.csv — 2,60,920 ball-by-ball records
